# Instalaciones
<p>1.- pip install virtualenv</p>
<p>2.- cd Aprendizaje_Animales</p>
<p>3.- py -3 -m venv venv</p>
<p>4.- pip install flask Flask-PyMongo</p>

# Ejecucion (Windows)
<p>1.- .\venv\Scripts\activate.bat</p>
<p>2.- python src/app.py</p>

# Ejecucion (Linux)
<p>1.- ./venv/Scripts/activate</p>
<p>2.- python src/app.py</p>
